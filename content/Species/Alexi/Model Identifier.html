Model Identifier

The Alexi’s model is composed by 3 parts like so XX-XXXXX-XXX
...

The model begins with its rank, followed by a 5 digit alphanumeric serial number, then variant (Which can be any short text, word, or abbreviation) eg.:

AA-564G8-Red, BA-63H91-GG, XB-42069-Wolf

The middle serial part is the model, not the individual. There may exist a few copies that have the same model number and look identical!

There are a few letters banned (not allowed for use) in serial numbers, because they look too similar to some numbers: B, I, O and Q

Serial Number
...

The serial number is only accessible internally. It follows the same restrictions as the model serial, but is composed of 25 digits instead of 5. It is assigned to individuals regardless of model.

The first 5 digits are assigned from the place of construction, the 6th digit is the control digit, and the rest is unknown.

Specialized Alexis
...

There might exist a few Alexis of an identical model, that were VR-trained to do different things. In order to differentiate them, you can add a Specialization Indicator, a 2-5 letter indicator before the serial number, that always ends on an S. (XS is reserved for Unspecialized)

It would look something like this:

XS-AA-0074A-Gold

X - no specialization
S - specialization indicator

This should only be used if an Alexi is a commercial model; meaning that it's widely produced for consumer purchase and not commissioned personally by someone, so that the same Alexi may be used for different things.

PRSM Mark
...

Prismas must be properly marked for their own safety. It goes in the "specialization" part of the model identifier, since those are not trained in any way and never use that slot. A Prisma could have a model ID like so:

PRSM-AA-0074A-BP
